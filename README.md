# WaitlistWealth – Email Collection Landing Page

## Monetization Strategy
1. Collect emails from interested users
2. Build a simple SaaS tool (e.g., AI resume reviewer, meeting summarizer, habit tracker)
3. Launch with a -20/month subscription
4. Convert 5% of your waitlist → monthly recurring revenue
